International Society of Automation, NSIT

Full Stack Developer

ISA-NSIT portal is designed specifically for the NSIT students aimed to automate the entire BTP timeline - from sending BTP proposals to faculties to tracking the progress of it. The portal acts as a medium for both students and faculties to make the BTP process more efficient and hassle free. The entire front end of the portal is designed simply using HTML5, CSS3, Bootstrap and Javascript. The backend part is developed using Ruby on Rails framework.

Zitter

Full Stack Developer

Zitter is based on the same idea as Twitter - "where users come to see whats happening around them". It was my first development project based on the Ruby on Rails framework. The front end code is written on plain HTML5, Bootstrap, CSS3 and Javascript.
